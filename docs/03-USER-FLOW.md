# MeetingMod - User Flow & Screen Specifications

## 1. 전체 사용자 흐름

```
┌─────────────────────────────────────────────────────────────────────────┐
│                           USER JOURNEY                                   │
└─────────────────────────────────────────────────────────────────────────┘

[0. 원칙 관리]    [1. 회의 준비]    [2. 회의 진행]    [3. 회의 종료]
      │                │                │                 │
      ▼                ▼                ▼                 ▼
┌──────────┐    ┌──────────┐     ┌──────────┐      ┌──────────┐
│ 원칙 편집  │    │ 준비 자료  │ ──> │ 회의 모드  │ ──>  │ 결과 화면  │
│   페이지   │    │   입력    │     │   진행    │      │   확인    │
└──────────┘    └──────────┘     └──────────┘      └──────────┘
      │                │                │                 │
      ▼                ▼                ▼                 ▼
 • Agile 원칙       • 아젠다 작성     • 실시간 자막      • 회의 요약
 • AWS LP 원칙      • 참석자 등록     • 화자 자동 분리    • Action Items
 • 커스텀 추가       • 원칙 선택       • 경고음+Toast     • Markdown 저장
```

### 3.2 컴포넌트 상세

#### 참석자 관리
- 이름 + 역할 입력
- 드래그로 순서 변경
- 참석자 삭제
- **중요**: 참석자 목록이 화자 분리에 사용됨

- 사전 정의된 원칙 템플릿 선택
- 복수 선택 가능
- 선택된 원칙은 Agent가 위반 감지에 활용

#### 아젠다 에디터
- Markdown 지원
- 실시간 미리보기

### 4.2 컴포넌트 상세

#### 헤더
- 회의 제목
- 녹음 상태 표시 (🔴 REC)
- 경과 시간

#### 실시간 자막 패널
- **화자명 자동 표시** (AI 화자 분리)
- 타임스탬프 + 발화 내용
- 스크롤 가능
- 새 발화 시 자동 스크롤
- Agent 개입 시 시각적 구분선

#### 발언 통계 패널
- 참석자별 발언 비율 (Progress Bar)
- 실시간 업데이트
- 불균형 시 경고 표시 (⚠️)

#### 아젠다 트래커
- 현재 논의 중인 아젠다 표시 (▶)
- 완료된 항목 체크 (✓)
- 클릭으로 수동 전환 가능

#### Agent 개입 패널 (핵심!)
- **경고음 재생** (🔔) - 시청각 주의 환기
- **Toast 형태로 메시지 표시**
- 개입 유형 표시
- [무시] / [확인] 버튼

### 5.2 컴포넌트 상세

#### 저장 완료 알림
- 저장된 디렉토리 경로 표시
- 성공 상태 표시

#### 회의 요약
- 기본 정보 (제목, 일시, 참석자)
- AI 생성 주요 논의 요약
- 3-5개 핵심 포인트

#### Action Items
- 추출된 Action Item 목록
- 담당자 + Due Date
- summary.md 및 action-items.md에 저장됨

#### 회의 품질 리포트
- 참여도 분포 차트
- 주제 집중도 점수
- Agent 개입 횟수

#### 개인별 피드백
- 탭으로 참석자 선택
- 좋았던 점 / 개선 제안 구분

- 생성된 Markdown 파일 목록
- [폴더 열기] 버튼으로 디렉토리 접근
