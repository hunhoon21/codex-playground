# AWS Leadership Principles for Meetings

1. **Customer Obsession**
   고객 관점에서 논의합니다.

2. **Ownership**
   책임감 있는 의견을 제시합니다.

3. **Disagree and Commit**
   이견을 표출한 후 결정을 따릅니다.

4. **Have Backbone; Disagree**
   동의하지 않으면 정중히 반박합니다.

5. **Dive Deep**
   세부사항까지 파악합니다.

6. **Bias for Action**
   빠른 결정, 실행 우선으로 진행합니다.
